简介(Introduction)
------------
swoft是基于swoole协程的高性能PHP微服务框架，内置http服务器。框架全协程实现，性能优于传统的php-fpm模式。

- 基于swoole易扩展
- 内置http协程服务器
- MVC分层设计
- 全局容器注入
- 高性能RPC
- 服务熔断、服务降级
- RPC连接池
- redis连接池
- mysql连接池
